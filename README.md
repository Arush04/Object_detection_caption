# Object_detection_caption  
The flask apps takes as input an image and returns a new image with all detected objects and also returns a json file with detected objects and a caption explaining the image. The json results are stored in the final_folder and the images are stored in the static folder.  
[View Image on Google Drive](https://drive.google.com/file/d/1sf_aB837dH3i5aR8e6syMNJFm9mN1F9j/preview)  
  
To Setup this project locally on your system:  
1. Git clone the repo.
2. cd into the directory and run the requirements.txt file (`pip install -r requirements.txt`).
3. Run the app2.py file.
